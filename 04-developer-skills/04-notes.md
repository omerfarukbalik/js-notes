## Prettier

Automatic code formatter. Install the VS Code extension, turn on **Format on Save**, stop thinking about indentation forever.

**Why it matters:** In a team, formatting arguments are a waste of everyone's time. Prettier ends the argument by removing the choice. Every real codebase has one.

---

## VS Code Shortcuts

| Shortcut | Action |
|---|---|
| `Ctrl + K` → `Ctrl + C` | Comment out selected lines |
| `Ctrl + K` → `Ctrl + U` | Uncomment selected lines |

## Habits Worth Keeping

- **Practice daily, in small blocks.** Consistency beats marathon sessions. Codewars is good for keeping the basics warm.
- **Rebuild array methods by hand.** Write your own `push`, `pop`, `indexOf` without using the built-in. Forces you to actually understand what they do instead of just calling them.
- **Read the error message.** Most beginners skip straight to Google. The message usually says exactly what's wrong.
